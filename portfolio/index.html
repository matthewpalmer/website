Programming

As well as Python for smaller scripts, I use Objective C to create iOS apps for modern smartphones.

DailyDo is an iOS app created to help people form better habits.

Using daily reminders, this app helps you form better habits by building up streaks and goals to share with friends. It features a bright and distinctive UI along with an experience designed to take the pain away from doing things every day.

Jotting.tk - the simplest notepad.

Jotting.tk is a dead simple note taking web app that autosaves everything you type. When you're done, you can export to a text file.
